# Last Ford (working title)
2D top-down strategy with procedural generation (Inspired by Stronghold).

## Current Features
- Procedural terrain with river division.
- Resource placement algorithms.
- Basic unit controller with A* pathfinding.

## Goals
- Build portfolio.
- Make a game of my dream.
- Experiment with clean architecture in Unity.
